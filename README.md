# BukUsaha V1.0

BukUsaha adalah aplikasi manajemen keuangan berbasis Laravel yang digunakan untuk mencatat pemasukan dan pengeluaran secara modern dan responsif.

---

## Fitur

- Dashboard Keuangan
- CRUD Transaksi
- Statistik Keuangan
- Grafik Pemasukan & Pengeluaran
- Filter dan Search Transaksi
- Pagination
- UI Modern Responsive

---

## Teknologi

- Laravel
- PHP
- MySQL
- Blade Template
- JavaScript
- Chart.js
- Tailwind CSS
- Laragon

---

## Instalasi

Clone repository:

```bash
git clone https://github.com/username/bukusaha.git
```

Masuk folder project:

```bash
cd bukusaha-app
```

Install dependency:

```bash
composer install
```

Copy file environment:

```bash
cp .env.example .env
```

Generate key:

```bash
php artisan key:generate
```

Migrasi database:

```bash
php artisan migrate
```

Jalankan server:

```bash
php artisan serve
```
